# RuntimeView

Basilico Plugin

## Required

* Basilico
    - [Basilico][Basilico]

* Qt5.7 or later
    - WebEngineWidget is required

[Basilico]:https://github.com/tsubota-kouga/Basilico

## Install
If you have dein.vim, you can write in your dein.toml

```toml
[[plugins]]
repo='tsubota-kouga/RuntimeView'
```
